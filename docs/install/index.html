How to install Tinyinfo

First download the ZIP File containing Tinyinfo. You can find the downloads here and here.

After you have downloaded Tinyinfo, extract the ZIP File into a folder of your choosing. Then right click the "Tinyinfo.exe" File and select Properties, if you see a checkbox saying "Unblock" please check it and press Ok.

Now you can launch the EXE and use Tinyinfo.
